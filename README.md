## &nbsp; Installation

Mechlife is an incredibly easy to use and setup.

Step 1: Go to the [releases](https://github.com/PDennSploit-Softworks-LLC/mechlife/releases/latest) and download the latest installer for your OS.

Step 2: Run the installer and follow the steps.

Step 3: Mechlife is now installed and your may use the app!

## &nbsp; Requirements

The only thing you have to install is the [Microsoft VC Redist 2015-2019 x64](https://aka.ms/vs/16/release/vc_redist.x64.exe) which is only around 100MB. This just allows the app to function correctly. If not Mechlife will be stuck on loading!
